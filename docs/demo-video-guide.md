- フォント: Noto Sans JP（Bold / Regular 使い分け）

## 2. 画面収録の手順（OBS Studio）

### 2-1. OBS インストール
1. https://obsproject.com/ja からダウンロード
2. インストール → 初回起動で「録画のために最適化」を選択

### 2-2. 録画設定

| 項目 | 設定値 |
|------|--------|
| 解像度 | 1920x1080（FHD） |
| FPS | 30 |
| 出力形式 | MP4（.mkv → 後で remux でもOK） |
| エンコーダ | x264 or NVENC（GPU搭載なら） |
| ビットレート | 6000 kbps |

**設定手順:**
1. 設定 → 映像 → 基本解像度: 1920x1080 / 出力解像度: 1920x1080
2. 設定 → 出力 → 録画 → 品質: 高品質 / エンコーダ: x264
3. 設定 → 出力 → 録画フォーマット: mp4

### 2-3. シーン・ソース設定
1. 「ソース」→「+」→「ウィンドウキャプチャ」→ ブラウザ（Chrome等）を選択
2. ブラウザは **全画面表示（F11）** にしてから収録開始
3. 不要なタブ・ブックマークバーは非表示に

### 2-4. 撮影ポイント（重要）

**撮影前の準備:**
- [ ] ブラウザのブックマークバーを非表示（Ctrl+Shift+B）
- [ ] 不要なタブをすべて閉じる
- [ ] ブラウザを全画面（F11）に
- [ ] 通知をOFF（Windows: 集中モード ON）
- [ ] デスクトップの壁紙は無地に（万が一映り込んだ時用）
- [ ] Invoice OCR にログインしておく
- [ ] ダミー請求書PDF（`public/demo/dummy-invoices-combined.pdf`）をデスクトップに置く

**撮影のコツ:**
- マウスはゆっくり動かす（視聴者が目で追える速度）
- クリック前に0.5秒ポーズを入れる（テロップを重ねる余白）
- ドラッグ＆ドロップは大きく動かす（小さいと見えない）
- 処理中のローディングアニメは全部映す（カットしない）
- 結果が出たら2〜3秒止まる（テロップ表示用）
- 失敗しても止めない。何テイクか撮ってベストを使う

**撮影フロー（この順番で操作）:**
1. Invoice OCR トップページを表示（2秒静止）
2. ダミーPDFをドラッグ＆ドロップでアップロード
3. AI処理中の画面をそのまま待つ
4. 結果一覧が表示されたら、ゆっくりスクロールして全件見せる
5. 「まとめてZIPダウンロード」をクリック
6. ダウンロードされたことを確認（ブラウザ下部のDL表示）
7. 2秒静止して録画停止

## 3. CapCut 編集ガイド

### 3-1. CapCut インストール
1. https://www.capcut.com/ja-jp/ からデスクトップ版をDL
2. インストール → 起動 → 「新しいプロジェクト」

### 3-2. プロジェクト設定
- アスペクト比: **16:9**（YouTube / LP埋め込み共通）
- 解像度: 1080p
- フレームレート: 30fps

### 3-3. 編集の流れ

#### STEP 1: 素材を読み込む
1. 録画した画面収録MP4をタイムラインにドラッグ
2. 不要な冒頭・末尾をトリミング（カット → Delete）

#### STEP 2: テロップを追加
1. タイムライン上部の「テキスト」→「テキストを追加」
2. 台本に沿ってシーンごとにテロップを配置
3. フォント: **Noto Sans JP** を選択
4. スタイル設定:
   - サイズ: 48〜64px（メインテロップ）/ 32px（補足テロップ）
   - 色: 白（#FFFFFF）
   - 背景: 黒 50%透過のバーを下に敷く（「スタイル」→「背景」ON）
   - 位置: 画面下 20% あたり
5. アニメーション: 「フェードイン」を選択（0.3秒）

#### STEP 3: 冒頭・末尾のスライド画面
1. 「テキスト」で課題提示テロップ（シーン1,2）を作成
2. 背景は「エレメント」→ グラデーション背景 or 単色黒
3. CTA画面（シーン9）も同様にテキストで作成

#### STEP 4: BGM を追加
1. 「オーディオ」→「音楽」→ CapCut内蔵のフリー音源を選択
2. おすすめジャンル: **Corporate / Inspiring / Light**（落ち着いたビジネス系）
3. 音量: **20〜30%**（テロップ主体なので控えめに）
4. フェードアウト: 最後の3秒で音量を0にフェード

#### STEP 5: トランジション
- シーン切替は「フェード」のみで統一（0.3〜0.5秒）
- 派手なトランジションは使わない（信頼感を損なう）

#### STEP 6: 書き出し
1. 右上「エクスポート」
2. 解像度: 1080p / フレームレート: 30fps
3. 形式: MP4
4. 品質: 「高」
5. ファイル名: `invoice-ocr-demo-v1.mp4`

### 3-4. YouTube アップロード時の設定（B用途の場合）
- タイトル: 「【税理士向け】請求書PDFの分割・命名を30秒で自動化｜Invoice OCR」
- 説明欄: サービスURL + 無料トライアルの案内
- タグ: 請求書, OCR, PDF分割, 税理士, 自動化, AI
- サムネイル: 「手作業30分→30秒」のビフォーアフター画像

## 4. AI音声ナレーション（オプション）

テロップだけでも成立するが、ナレーションを入れたい場合:

### VOICEVOX（無料・商用OK）
1. https://voicevox.hiroshiba.jp/ からDL
2. 話者おすすめ: **四国めたん（ノーマル）** or **ずんだもん（ノーマル）**
   - ビジネス寄りなら四国めたん、親しみやすさならずんだもん
3. 台本テキストを貼り付け → 音声生成 → WAVで書き出し
4. CapCut に WAV を読み込んで、テロップと同期させる
